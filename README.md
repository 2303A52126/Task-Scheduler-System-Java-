# 🗂️ Task Scheduler System (Java)

## 📌 Overview

This project is a console-based **Task Scheduler System** built using Java. It simulates how real-world systems like operating systems and backend services manage and execute tasks efficiently based on priority.

The system allows users to add, manage, and execute tasks in an optimized manner using efficient data structures.

---

## 🚀 Features

* Add tasks with priority
* View all scheduled tasks
* Execute tasks based on priority
* Delete tasks
* Menu-driven console interface
* Efficient scheduling using Priority Queue

---

## 🧠 Tech Stack

* **Language:** Java
* **Concepts:** Data Structures & Algorithms, OOP
* **Data Structure Used:** Priority Queue (Heap)

---

## ⚙️ How It Works

* Tasks are stored in a **Priority Queue**
* Higher priority tasks are executed first
* Internally uses heap-based logic for efficient scheduling

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/task-scheduler-system.git
   ```

2. Navigate to project folder:

   ```bash
   cd task-scheduler-system
   ```

3. Compile:

   ```bash
   javac Main.java
   ```

4. Run:

   ```bash
   java Main
   ```
